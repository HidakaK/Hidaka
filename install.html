SC-03Eの場合なので、他の機種だと異なる可能性があリます。
  1. 端末にROMとgApps,magiskのzipファイルをコピーする。
  2. ダウンロードしたOdinを起動する。
  3. 端末を電源ボタン、ボリュームダウンボタン、ホームボタンの同時押しで起動させる。警告画面の後、バグドロイドが表示された画面になると成功
  4. PCに接続し、odinのmessageにadded!!が表示されたらウィンドウ上のPDA(ver3.1?以降はAP)をクリックし、カスタムリカバリのファイルを選択する。拡張子がtar.md5の時は.tarにリネームする
    また、ここでストックROMを選択することでオリジナルの状態にすることもできる。 その後、startをクリックする。緑でPASS!と表示されれば完了。
  6. PCから抜き、電源、ホーム、ボリュームアップボタンをdocomoのロゴが表示されるまで押し続ける。その後TWRPが起動するので、wipeからdalvik,cache,system,dataをwipeし、installでromのzipファイルとopengapps,magiskのファイルを選択肢、インストールする。
